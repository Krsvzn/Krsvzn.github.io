How to Install

Since this app is distributed directly, you'll need to enable installation from unknown sources.

1

Download the APK

Click the download button above to get the latest version of Gate Icon.

2

Open the File

Tap on the downloaded APK file. Your phone may show a security warning.

3

Allow Installation

Go to Settings → Security → Install unknown apps and enable permission for your browser or file manager.

4

Install & Open

Tap "Install" and wait for the process to complete. Then open Gate Icon and sign in!

Pro Tip: After installation, you can disable "Install unknown apps" for additional security. The app will notify you when updates are available.
